# 介绍


一款告警路由转发系统，支持灵活的告警路由规则，并能够对告警进行聚合收敛，减少告警数量。
## 告警对接
{{< hint info >}}
- [x] 支持常用的客户端对接
- [x] 可以基于JavaScript实现自定义对接和逻辑
{{< /hint >}}
